# 💰 Split Expense Calculator

A simple Python CLI project that helps users calculate their total bill including tip and GST, and optionally split the bill among friends.

---

## 🚀 Features

* Calculate total bill amount
* Add optional tip (in percentage)
* Automatically include **18% GST**
* Split bill among multiple people
* Clean and user-friendly command-line interface

---

## 🧠 How It Works

1. User enters the bill amount
2. Chooses whether to add a tip
3. Tip is calculated as a percentage of the bill
4. GST (18%) is added automatically
5. Final total is displayed
6. Option to split the bill among friends

---

## 🛠️ Technologies Used

* Python (Beginner Level)
* Input/Output handling
* Conditional statements (`if-else`)
* Basic arithmetic operations

---

## ▶️ How to Run

1. Install Python (if not installed)
2. Download or clone this repository
3. Run the script:

```bash
python split_expense_calculator.py
```
