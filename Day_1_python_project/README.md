# 🚀 Username Generator (Python)

A simple Python project that automatically generates multiple username/email combinations based on user input.

---

## 📌 Project Description

This project takes basic user details like:

* First Name
* Last Name
* Year of Birth

…and generates multiple username/email suggestions in a structured format.

It is a beginner-friendly project focused on:

* User input handling
* String manipulation
* Formatting output using f-strings

---

## 🛠️ Features

✅ Generates multiple username combinations
✅ Uses string indexing and formatting
✅ Clean and readable output
✅ Beginner-friendly logic

---

## 💻 How It Works

The program:

1. Takes input from the user
2. Combines strings in different formats
3. Displays auto-generated usernames

---

## 📷 Example Output

```
Please enter your name: Vasu
Please enter your last name: Rana
Please enter your birth year : 2004

###  Here is your auto_generated Usernames   ###

1-> VasuRana2004@xyz.com
2-> VRana2004@xyz.com
3-> Vasu2004@xyz.com
```

---

## 🧠 Concepts Used

* `input()` function
* String concatenation
* String indexing (`first_name[0]`)
* f-strings
* Basic Python syntax

---

## ▶️ How to Run

1. Make sure Python is installed
2. Save the file as `username_generator.py`
3. Run the script:

```
python username_generator.py
```
